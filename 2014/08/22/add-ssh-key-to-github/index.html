为Github添加SSH KEY

发表于 2014-08-22   |   分类于 Ops   |  

如果你使用Github，Bitbucket, 或者国产的Coding.net等，都会在个人设置那里看到设置SSH Key的地方，这个是做什么用的呢，这个是用来让你提交代码通过SSH隧道的方式提交到远程服务器，这种方式有安全，稳定以及无需每次输入认证信息等特点。

本文只是以Github为例，讲解怎么操作，至于原理和概念，请自行搜索。
